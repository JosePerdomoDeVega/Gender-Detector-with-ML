# Gender Detector

Predicts the apparent gender of a face in a photo. The pipeline crops the face, turns it into a
**Facenet512 embedding**, and classifies it with a scikit-learn model trained on ~1,000 labelled
CelebA faces.

This was one of my first machine-learning projects. I've since refactored it from a single
Jupyter notebook into a small, runnable Python package with a web interface, a reproducible
training step and tests.

## Demo

Run the app and either upload a photo or take one with your webcam:

```bash
python app.py
```

It opens a local Gradio interface. The result includes the predicted label and, for models that
expose probabilities, a confidence score.

## How it works

```
face image  ->  DeepFace (Facenet512)  ->  512-dim embedding  ->  sklearn classifier  ->  label
```

- **Face → embedding.** [DeepFace](https://github.com/serengil/deepface) detects and crops the
  face and produces a 512-dimensional Facenet512 embedding.
- **Embedding → label.** A classifier maps the embedding to `male` / `female`. Training compares
  Logistic Regression, a Perceptron and an SVC on a held-out test set and keeps the best one.

On the CelebA test split the SVC reaches **93.69%** accuracy.

## Project layout

```
app.py                     Gradio interface (upload or webcam)
src/gender_detector/
  config.py                paths and embedding settings
  embedder.py              image -> Facenet512 embedding (lazy DeepFace import)
  train.py                 compare models, keep the best, save to models/
  predictor.py             load the saved model and predict
test/                      training + predictor tests (run on the precomputed vectors)
Images-CelebA-1K/          precomputed embeddings + labels (train/test)
models/                    trained classifier (.joblib, generated by train.py)
```

## Setup

```bash
python -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Train and save the classifier (reads the precomputed embeddings in `Images-CelebA-1K/`):

```bash
python -m src.gender_detector.train
```

Then launch the app:

```bash
python app.py
```

or 

```bash
python app
```


## Tests

```bash
pytest
```

The tests run against the precomputed embedding CSVs, so the core training/prediction logic is
verified without needing DeepFace or a webcam.

## A note on the model and its limits

This is a demonstration project. The classifier predicts a binary, **apparent** gender learned
from the CelebA celebrity dataset, which is not necessarily diverse or balanced. It should not be used to make decisions about real people.
